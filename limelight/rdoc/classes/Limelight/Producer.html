Class Limelight::Producer
In: lib/limelight/producer.rb
Parent: Object

A Producer has the hefty responsibility of producing Productions. Given a directory, it will load the neccessary files and create all the neccessary objects to bring a Production to life.

For directory structures, see Limelight::Main
