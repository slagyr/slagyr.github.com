Statemachine Gem

This Ruby Library enables simple creation of full-features Finite Statemachines.

API

Where to start:

  • Statemachine.build
  • Statemachine::SuperstateBuilding
  • Statemachine::StateBuilding
  • Statemachine::Statemachine

Documentation

Some documentation is available here in this RDOC documentation. You may also find useful documentation on the Statemachine website: statemachine.rubyforge.org

A detailed tutorial and overview of Finite State Machines and this library can be found at blog.8thlight.com/articles/2006/11/17/understanding-statemachines-part-1-states-and-transitions.
